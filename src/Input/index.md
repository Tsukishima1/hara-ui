# Input 输入框

_该组件封装了原生的`input`标签，提供了更多的样式和功能。_

输入框组件用于接收用户输入的文本信息。

## 引用

```jsx | pure
import { Input } from 'hara-ui';
```

| 参数      | 说明       | 类型                  | 默认值 |
| --------- | ---------- | --------------------- | ------ |

| className | 自定义类名 | `string`              | -      |
