## Development

```bash
# install dependencies
$ npm install

# develop library by docs demo
$ npm start

$ npm run build

# build library source code in watch mode
$ npm run build:watch

# build docs
$ npm run docs:build

# Locally preview the production build.
$ npm run docs:preview

# check your project for potential problems
$ npm run doctor
```
